ReInstatement Transaction Types :

Operation Name Request and Response Operation Description
CancellationTransaction
CancellationRequest
CancellationResponse
ReInstatement Transaction Type : CN00

Description :

The CN00 screen is used to calculate fees with compliance codes and dates. Once the compliance code is set, fees automatically generate for the dates set. Be sure the compliance code and date are correct before proceeding. The CN00 screen will automatically roll over from the LS00 screen.

DL00TransactionRequest
DL00TransactionResponse

Description :

To remove the NI flag

DriverLicenseHistory
DriverLicenseHistoryRequest
DriverLicenseHistoryResponse
ReInstatement Transaction Type : DL00

Description :

To retrieve the customers drivers license information

ClearanceLetter
ClearanceLetterRequest
ClearanceLetterResponse
ReInstatement Transaction Type : 2000

Description :

Once compliance is received for an NI flag and is cleared, a clearance letter is given to the customer indicating the cancellation have been cleared and the money received (if any) for the clearance. This letter must be printed on state letterhead. All cancelations cleared must be on separate clearance letters. If the customer has not paid the full fee, only partial, utilize the FE00 screen printout for the receipts of the money collected
LicensePlate
LicensePlateRequest
LicensePlateResponse
ReInstatement Transaction Type : 11LP

Description :

The vehicle record screen is used to remove the NI flag from vehicle records and enter the vehicle deposition information if applicable. The NI flag can only be removed from a record once all flags for that vehicle have been paid or cleared
